Hello there!

My name is Jeroen, and I work as a front-end web developer out of Amsterdam. I like to build beautiful and often minimalistic websites, with a particular interest in using latest techniques and design methodologies.

This wasn't always the case though. It was in the beginning of 2017, while being employed in my long time job as Trademark Attorney, when I came across an accelerated Code Academy program to learn web development within a matter of months. Since I strongly felt it was time for a new and exciting direction, I decided to quit my job and take on that challenge.

It's been a very fun and rewarding experience so far.

What else?

So what do I like to build with? I especially enjoy working on projects that are based on ReactJS or WordPress. The former because it allows for comprehensive interactive web applications and 'feels' like the future of development, the latter because it creates highly user-friendly websites that are fun to make and easily maintainable for a client.

Since this website itself is pretty concise, it doesn't really require the use of a big framework or CMS and for this reason is build from scratch using HTML, CSS and some Javascript. The CSS is written by using PostCSS as a (pre)processor with the Preset Env plugin to enable future CSS syntax. Babel and Webpack take care of the compiling and building of the Javascript, and Gulp handles the automation of all these tasks during development.
