Hello there!

My name is Jeroen, and I work as a front-end developer out of Amsterdam. I like to build beautiful and often minimalistic websites, using the latest techniques and design methodologies.

This wasn't always the case though. In the beginning of 2017, when I was employed in my long time job as Trademark Attorney, I came across an accelerated Code Academy program to learn web development within a matter of months. Since I strongly felt it was time for a totally new and different challenge, I decided to quit my job and take the plunge. So far, it's been a very interesting ride :-)

What else?

So what do I like to build with? I especially enjoy working on projects that are based on ReactJS or WordPress. The former because it allows for comprehensive interactive web applications and 'feels' like the future of development, the latter because it creates highly user-friendly websites that are fun to make and easily maintainable for a client.

Since this website is pretty lightweight and straightforward, it doesn't use a framework and is build from scratch using HTML, PostCSS as a CSS (pre)processor with the Preset Env plugin to enable the use of future CSS syntax and Babel and Webpack to compile and bundle the Javascript. To automize these tasks during development I'm using Gulp.
